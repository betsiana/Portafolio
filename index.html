PROYECTOS

Laboratoria News

Proyecto final Sprint 6.

La página Laboratoria News aspira a ser la primera opción y referencia editorial para los peruanos.

Para la creación de esta página se uso Html5, Sass, gulp, jquery y un archivo json de donde obtendremos las noticias a mostrar y nombres de las fotos de la noticia.

Yape

Proyecto final Sprint 5.

La app Yape del BCP te brinda la libertad de usar tu dinero como quieras, ya que pagar con ella es más rápido.

Para la creación de la replica de la app se uso Html5, jquery y un json en donde se envio data y de esta manera se obtuvo codigos de confirmación.

LabCar | Servicio de taxi

Code review de proyecto final Sprint 4.

La página web LabCar ofrece el servicio de taxi, da la ubicación actual del usuario y le permite ingresar punto de salida y llegada para trazar la ruta.

Para la creación de esta pagina se uso Html5, Css3, jquery y el API de googlemaps donde obtendremos el autocompletado y trazo de ruta.

Pinterest

Proyecto grupal sprint 6.

La página de Pinterest es una plataforma para compartir imágenes que permite a los usuarios crear y administrar tableros personales.

Para la creación de esta replica se uso Html5, sass, jquery, gulp y el API de Pinterest donde obtendremos las imagenes y datos necesarios.
